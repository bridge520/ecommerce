# 電商網站設計文件

**日期：** 2026-04-14
**版本：** 1.0

---

## 1. 專案概述

### 目標
建立一個小型工作室電商網站，支援實體商品與數位商品的混合銷售，面向台灣及國際市場。

### 規模定位
- 小型個人工作室 / 賣家
- 商品數量 < 100
- 支援多語言（繁體中文、英文、日文）

---

## 2. 技術架構

### 建置方案
**方案 A：標準 WooCommerce + 精選插件組合**

### 系統層次

```
主機層        VPS 或台灣主機商（Nginx + PHP 8.x + MySQL）
核心平台      WordPress 6.x + WooCommerce 8.x
功能插件      多語言 / 金流 / 物流 / 數位商品 / SEO / 速度優化
前台主題      客製化 WordPress 佈景主題（Astra 或 Flatsome）
```

### 主機選擇
- **推薦：台灣主機商**（速博、網路中文、伊雲谷）— 月費約 NT$300–600，台灣 IP、中文客服
- **備選：國際 VPS**（DigitalOcean、Vultr）— 月費約 $12–24 USD，彈性較高

---

## 5. 購物流程

### 實體商品
```
瀏覽商品 → 加入購物車 → 填寫收件資訊 → 選擇物流方式
→ 選擇付款方式 → 付款成功 → 出貨通知信（自動）
```

**物流選項：**
- 7-11 / 全家超商取貨（ECPay 物流）
- 黑貓宅配
- 自取

### 數位商品
```
瀏覽商品 → 加入購物車 → 結帳付款 → 付款成功
→ 信箱自動寄送下載連結 + 會員帳號頁可下載
```

**數位商品保護：**
- 下載連結設定過期時間
- 限制下載次數

## 7. 多語言規格

- **支援語言：** 繁體中文（預設）、英文、日文
- **實作方式：** WPML 插件，每個商品、頁面各語言版本獨立管理
- **SEO：** 各語言頁面使用 hreflang 標記（Yoast SEO + WPML）
- **前台：** 語言切換器放置於導覽列

---

## 8. 效能與安全

### 效能
- WP Rocket 頁面快取
- Cloudflare CDN（免費方案）圖片與靜態資源加速
- 商品圖片使用 WebP 格式壓縮

### 安全
- SSL 憑證（Let's Encrypt 免費）
- WordPress 定期更新
- 限制後台登入嘗試次數（Limit Login Attempts 插件）
- 定期資料庫備份
